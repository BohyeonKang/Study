##4장 Index와 Join

###4.3 JOIN

####Nested Loop Join

* Index 상황
```
* PK_DELP        : DEPT.DEPTNO
* DEPT_LOC_IDX   : DEPT.LOC
* PK_EMP         : EMP.EMPNO
* EMP_DEPTNO_IDX : EMP.DEPTNO
* EMP_SAL_IDX    : EMP.SAL
```

* Query
```SQL
SELECT /*+ ORDERED USE_NL(e) */ E.EMPNO, E.ENAME, D.DNAME, E.JOB, E.SAL
  FROM DEPT d, EMP e
 WHERE e.DEPTNO = d.DEPTNO    -- #1
   AND d.LOC    = 'DALLAS'    -- #2
   AND e.COMM IS NULL         -- #3
   AND e.SAL   >= 1500        -- #4
 ORDER BY SAL DESC
```
